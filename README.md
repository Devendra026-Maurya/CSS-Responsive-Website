# 🌐 CSS Responsive Projects Collection

This repository contains a set of fully responsive website projects built using **HTML5**, **CSS3**, and modern layout techniques such as **Flexbox**, **CSS Grid**, and **Media Queries**.

These projects were created to practice responsive web design and improve my frontend development skills.

---

## 📁 Projects Included

### 11. Food Restaurant Website
A responsive restaurant website featuring:
- Hero section
- About section
- Menu showcase
- Customer reviews
- Contact section

### 12. Flower Shop Website
A beautiful responsive website for an online flower shop with:
- Product cards
- Categories
- Testimonials
- Newsletter section

### 13. Responsive Grocery Store
A modern grocery store website featuring:
- Product listings
- Shopping categories
- Special offers
- Responsive navigation

### 14. Delta Ecommerce Website
A complete responsive eCommerce website with:
- Homepage banner
- Product sections
- Shopping cart UI
- Newsletter and footer
